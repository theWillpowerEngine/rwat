Full Command List

Movement

W/A/S/D Normal movement
Shift+W/A/S/D Diagonal movement (45 degrees counter-clockwise)

Actions and Interactions

L Toggle personal lantern
T Toggle turning controls up/down
C Show Command Menu to command crew

Mappable Actions / Special Interactions

1/2/3/4/5 Special Actions 1-5
Q/E/R/F/G Special Actions 6-10
Shift+1/2/3/4/5 Special Actions 11-15
Shift+Q/E/R/F/G Special Actions 16-20

UI

Shift-C Show Character Sheet
Tab Switch between view layers/maps
Alt-Enter Toggle Fullscreen mode

Game System

Ctrl-S Save Game (one save slot only)
Ctrl-O Load Current Save
Esc Exit game

Game Time

Space Advance 1 tick without doing anything
Shift-Space Advance 5 ticks
Ctrl-Space Advance 10 ticks
Sh-Ctrl-Space Advance 25 ticks
P Unpause or Pause game
+/- Speed/Slow time. Zoom on World Map
